
# muthu-footer

A simple reusable sticky footer for Streamlit apps developed by M. Kumaran.

## Installation

```bash
pip install muthu-footer
```

## Usage

```python
from muthu_footer import add_footer

# Your Streamlit app code...

add_footer()
```
